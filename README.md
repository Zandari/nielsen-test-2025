## Установка зависимостей

Для Linux/MacOS:
```bash
python -m venv venv
source venv/bin/activate
pip install -r requirements.txt
```

Для Windows:
```bash
python -m venv venv
venv\Scripts\activate.bat
pip install -r requirements.txt
```

## Запуск notebook'а

> Подразумевается что вы находитесь в виртуальном окружении

```bash
jupyter notebook
```
